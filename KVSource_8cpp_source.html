14 // Simple source reconstruction global variable
16 //
17 // It takes all nuclei selected with SetSelection(KVParticleCondition&) and adds them
18 // together (charge, mass, momentum, energy) in order to calculate
19 // a 'source' nucleus, for which we can obtain various quantities (see below).
20 // The reference frame from which particle momenta are taken can be set with SetFrame("..").
21 // Each nucleus which is included in the source definition is added
22 // to a group with the same name as this variable. This can be used for further analysis
23 // of the particles (by testing KVParticle::BelongsToGroup("...") method).
24 //
25 // Index Name Meaning
26 //-----------------------------------------------------
27 // 0 Ex Excitation Energy
28 // 1 Z Z source
29 // 2 A A source
30 // 3 Vz z component of velocity
31 // 4 Vx x component of velocity
32 // 5 Vy y component of velocity
33 // 6 Theta polar angle relative to the beam
34 // 7 Mult number of nuclei added to source
35 //
36 // These quantities can be obtained either by name:
37 //
38 // Double_t estar = GetValue("Ex");
39 //
40 // or by index:
41 //
42 // Double_t estar = GetValue(0);
43 //
44 // or in an array:
45 //
46 // Double_t *vals = GetValuePtr();
47 // Double_t estar = vals[0]
48 //
