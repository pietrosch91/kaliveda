16 // Global variable returning the ZVtot variable. It is defined as follow:
17 //
18 // ZVtot= Sum(Z(i)*v(i))
19 //
20 //
21 //with v(i) : velocity of particle i; (in cm/ns)
22 // Z(i) : charge of particle i;
23 //
24 //The units of ZVtot are therefore [cm/ns].
25 //
26 //By default ZVtot is calculated from particle velocities in the default (laboratory) frame.
27 //However, this can be changed by giving the name of the desired frame
28 //to either the constructor
29 // KVZVtot* zvtot = new KVZVtot("my_ZVtot", "CM"); // calculate in "CM" frame
30 //or by using KVVarGlob method SetFrame():
31 // zvtot->SetFrame( "QP" ); // calculate in "QP" frame
32 //
33 // The names of the variables and the corresponding indexes are as follows:
34 //
35 // Name Index Meaning
36 //----------------------------------------------------------------------------
37 // Z 0 (default) Z component of ZVtot
38 // X 1 X component of ZVtot
39 // Y 2 Y component of ZVtot
40 //
41 //
42 // All these values can be obtained by calling the GetValuePtr() method which
43 // returns an array of Double_t containing the values.
44 //
45 // The vector (TVector3) can be used by calling the GetTVector3() method.
46 //
47 // If you want to normalize the values e.g. to the product Z*v of the projectile
48 // in the laboratory frame, set the parameter "Normalization"
