11 <h2>KVIdentificationResult</h2>
12 <h4>Full result of one attempted particle identification</h4>
13 When we attempt to identify a reconstructed charged particle (<a href="KVReconstructedNucleus.html">KVReconstructedNucleus</a>) using an identification
14 telescope (<a href="KVIDTelescope.html">KVIDTelescope</a>), we call the
15 KVIDTelescope::Identify(KVIdentificationResult*) method of the telescope in question, and the results of the
16 identification attempt are stored in the KVIdentificationResult object.
17 <br><br>
18 The informations stored are:
19 <ul>
20 <li>Bool_t IDattempted : (=kTRUE if an identification attempt was made)</li>
21 <li>Bool_t IDOK : general status of identification (=kTRUE if identification attempt successful)</li>
22 <li>GetIDType() : returns string containing type of identification (corresponds to type of identification telescope)</li>
23 <li>Int_t IDcode : general identification quality code associated with this type of identification</li>
24 <li>Int_t IDquality : specific quality code returned by identification procedure</li>
25 <li>GetComment() : explanatory message regarding quality code returned by identification procedure</li>
26 <li>Double_t PID : the particle identifier (interpolated mass or charge)</li>
27 <li>Bool_t Zident : (=kTRUE if particle's Z was determined by the identification)</li>
28 <li>Bool_t Aident : (=kTRUE if particle's A was determined by the identification)</li>
29 <li>Int_t Z : the Z given by the identification attempt if Zident==kTRUE</li>
30 <li>Int_t A : the A given by the identification attempt if Aident==kTRUE</li>
31 <li>Int_t deltaEpedestal : tells if particle is in pedestal region of delta-E</li>
32 </ul>
33 Note that, apart from GetIDType() and GetComment() methods, all other informations are public member variables
34 which can be accessed directly.
35 <!-- */

118 void KVIdentificationResult::Print(Option_t*) const
119 {
120  printf("Identification #%d - Type:%s ", GetNumber(), GetIDType());
121  if (!IDattempted) {
122  printf(" => not attempted\n\n");
123  return;
124  }
125  if (IDOK) printf(" => SUCCESS\n");
126  else printf(" => FAILURE\n");
127  printf(" Quality code = %d (%s)\n", IDquality, GetLabel());
128  if (Zident) printf(" Z identified = %d", Z);
129  else printf(" Z returned = %d", Z);
130  if (Aident) printf(" A identified = %d", A);
131  else printf(" A returned = %d", A);
132  if (Zident || Aident) printf(" PID = %f\n", PID);
133  printf(" delta-E pedestal : ");
134  switch (deltaEpedestal) {
135 
136  case deltaEpedestal_NO:
137  printf("NO\n");
138  break;
139 
140  case deltaEpedestal_YES:
141  printf("YES\n");
142  break;
143 
144  default:
