12 ClassImp(KVPtot)
14 // Global variable returning the Ptot variable. It is defined as follow:
15 //
16 // Ptot= Sum(p(i))
17 //
18 //
19 //with p(i) : momentum of particle i;
20 //
21 //By default Ptot is calculated from particle momenta
22 //in the default (laboratory) frame.
23 //However, this can be changed by giving the name of the desired frame
24 //to either the constructor
25 // KVPtot* ptot = new KVPtot("my_Ptot", "CM"); // calculate in "CM" frame
26 //or by using KVVarGlob method SetFrame()
27 // ptot->SetFrame( "QP" ); // calculate in "QP" frame
28 //
29 // The names of the variables and the corresponding indexes are as follows:
30 //
31 // Name Index Meaning
32 //----------------------------------------------------------------------------
33 // Z 0 (default) Z component of Ptot
34 // X 1 X component of Ptot
35 // Y 2 Y component of Ptot
36 //
37 //
38 // All these values can be obtained by calling the GetValuePtr() method which
39 // returns an array of Double_t containing the values.
40 //
41 // The vector (TVector3) can be used by calling the GetTVector3() method.
42 //
43 // If you want to normalize the values e.g. to the momentum of the projectile
44 // in the laboratory frame, set the parameter "Normalization"
45 Int_t KVPtot::nb = 0;

47 Int_t KVPtot::nb_dest = 0;
