void BackTrack::GenericModel_Binned::SaveInitWorkspace ( char *  file = 0)

To save an initial worksapce for further fits Will save the needed objects for the performing another fit without generating the new RooDataSets (can be long) will contain: _parameters = RooArgList of the parameters _obsevables = RooArgList of the observables _parobs = RooArgList of the param+observables (mandatory for cuts in the RooDataHist) _datahistset = TObjArray of the model RooDataHist _datasetparams = TObjArray of the model RooDataHist parameters This workspace can then be used with SetWorkspace() method By default not saved and saved in a file which name will contain observables informations If workspace already provided, will not save it again
