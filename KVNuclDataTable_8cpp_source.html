18 General methods:
19 <ul>
20  <li>Bool_t IsInTable(Int_t zz, Int_t aa) - returns kTRUE for nuclei which appear in

22 
23 Specific implementations must define the methods:
24 <ul>
25  <li>Initialize() - called once to initialize mass table, for example by
26  reading in values from a file</li>
27  <li>Double_t GetValue(Int_t zz, Int_t aa) - returns value of the tabulated quantity</li>
28  <li>CreateTable(Int_t ntot) - Create table where for each nucleus, a pointeur of the class will be stored</li>
29  <li>CreateElement(Int_t idx)- Create element which correspond to the class pointeur associated to the nucleus</li>
30  For example :
31  If we consider a class KVLifeTime where is stored the value of life time of a nucleus
32  _____________
33  void KVNuclDataTable::CreateTable(Int_t ntot)
34  {
35  theTable = new KVLifeTime*[ntot];
36  }
37  __________________
38  void KVNuclDataTable::CreateElement(Int_t idx)
39  {
40  current_idx = idx;
41  theTable[idx] = new KVLifeTime();
42  }
43  For further detail see the KVLifeTimeTable and KVLifeTime class
44 </ul>
