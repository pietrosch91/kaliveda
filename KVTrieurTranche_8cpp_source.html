19 // ***** Class for a sorting with attached cells. ******
20 //
21 // This class returns an index determined via a a research in an array of
22 // values. The upper limit of a cell in the array is the lower limit of the next
23 // cell. The following methods are provided by this class:
24 // virtual Int_t GetNumCase(Double_t x) Gives an index corresponding to
25 // the value of x. If xarray is the
26 // array of nb_cells values, the
27 // returned value i is determined
28 // following this prescription:
29 //
30 // xarray[i-1] < x < xarray[i]
31 //
32 // if x<x[0] it returns -1
33 // if x>x[nb_cells] it returns -1
34 // -1 means "out of range"
35 //
36 // virtual Int_t GetNumCase(void *,...) return -1,not to use.
37 //
38 // BEWARE! : the index value ranges between 1 and nb_cells.
39 //
40 // For each index, the Xmin and Xmax values can be adjusted with the methods
41 // virtual void SetXmin(Int_t i,Double_t x)
42 // virtual void SetXmax(Int_t i,Double_t x)
43 //
44 // BEWARE! : the index value ranges between 1 and nb_cells.
45 // Remember that calling SetXmax(i,x) is equivalent to call SetXmin(i+1,x), or
46 // that calling SetXmin(i,x) is equivalent to call SetXmax(i-1,x).
47 //
48 // The name of the sorting variable can be set withe the method
49 // virtual void SetNomVar(Char_t *s)
50 //
51 // The number of indexes can be set with the method
52 // virtual void SetNbCases(Int_t n)
53 //
54 // Setting the number of indexes, the name of the sorting variable or the Xmin
55 // or Xmax values automatically generates the names for each index.
56 //
57 // In a treatment program, this can be used to set histogram titles and to
