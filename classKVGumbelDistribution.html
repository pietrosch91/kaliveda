KVGumbelDistribution Class Reference

Gumbel distributions for rank-ordered extremal variables.

Consider a probability distribution f(Z) which is used to generate m values of the random variable Z, which can then be ordered in terms of decreasing size

BEGIN_LATEX Z_{1}>Z_{2}> ... >Z_{m} END_LATEX

If such a procedure is repeated a sufficiently large number of times, it can be shown that the resulting probability distributions of the Z_k are given by the Gumbel distributions:

BEGIN_LATEX {k}(s_{k}) = {k^{k}}{(k-1)!}{1}{b_{m}}e^{-ks_{k}-ke^{-s_{k}}} END_LATEX

where

BEGIN_LATEX s_{k} = {Z_{k}-a_{k}}{b_{k}} END_LATEX
