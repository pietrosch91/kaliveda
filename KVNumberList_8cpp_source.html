13 <h2>KVNumberList</h2>
14 <h4>Strings used to represent a set of ranges of values</h4>
15 
16 Handles lists of positive numbers such as `"15-17 1-12 14"` or `"345,356-390"`.
17 Strings with this format can be handled and analysed by this class.
18 They can also be iterated over. Note that the numbers in the initializing string
19 do not have to be in ascending order, or indeed unique. The resulting KVNumberList
20 will contain only the unique values in ascending order.
21 
22 ### Examples
23 
24 Create a new list with the constructor:
25 
26  KVNumberList a("1-20, 51, 52-56")
27 
28 Change an existing list:
29 
30  a.SetList("1001 1003-1005,1010")
31 
32 Get first and last values of ranges in list (i.e. largest and smallest included values)
33 
34  a.First() (would give 1001 for previous example)
35  a.Last() (would give 1010 for previous example)
36 
37 Get an array filled with all values corresponding to ranges defined in list:
38 
39  Int_t n
40  Int_t* val = a.GetArray(n)
41 
42 `val[0]` is same value as `a.First()`.
43 
44 `val[n-1]` is same value as `a.Last()`.
45 
46 Use AsString() or GetList() method to obtain list in its most compact form, using the "1-20, 51, 52-56" format.
47 
48 IsEmpty() returns kTRUE if the list is empty ;-p
49 
50 #### Iterating over all numbers in the list
51 
52 Initialise first by calling Begin(), then loop until End() returns kTRUE:
53 
54  KVNumberList r("1-10");
55  r.Begin();
56  while( !r.End() ){
57  Int_t next_val = r.Next();
58  ...
59  }
60 
61 If list is empty, End() always returns kTRUE and Next() returns -1.
62 */

499 IntArray KVNumberList::GetArray() const
500 {
501  // Creates and fills a sorted array with all the unique
502  // values compatible with the ranges defined in the list.
503  // (Sorting is in increasing order).
504 
505  if (IsEmpty())
506  return IntArray();
507 
508  if (!fIsParsed) ParseList();
509 
510  IntArray temp(fNValues);
511  Int_t index = 0;
512  for (int i = 0; i < fNLimits; i++) {
513  Int_t min = (*fLowerBounds)[i];
514  Int_t max = (*fUpperBounds)[i];
515  for (int j = min; j <= max; j++) {
516  temp[index++] = j;
517  }
518  }
519  //now check for duplicate entries
520  //we sort the array in increasing order
521  //any duplicate entries will then be adjacent
522  IntArrayIter beg = temp.begin();
523  std::sort(beg, temp.end());
524  IntArrayIter end = std::unique(beg, temp.end());
525  Int_t n_uniq = std::distance(beg, end); //number of unique values
526 
527  if (n_uniq < fNValues) {
528  // duplicates were removed
529  // reduce the size of the vector
530  temp.resize(n_uniq);
531  // we reconstruct a string containing all unique values & reparse it
532  fString.Form("%d", (*(beg++)));
533  while (beg != end) {
534  fString += Form(" %d", (*(beg++)));
535  }
536  fNValues = n_uniq;
537  fIsParsed = kFALSE; // force re-parsing
538  ParseList();
539  }
540  return temp;
541 }
