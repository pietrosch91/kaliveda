21 <h2>KVSeqCollection</h2>
22 <h4>KaliVeda extensions to ROOT collections</h4>
23 This class adds functionalities such as
24 <ul>
25 <li>FindObjectByType</li>
26 <li>FindObjectByLabel</li>
27 <li>GetSublistWithType</li>
28 </ul>
29 etc. to the standard ROOT collection classes. The actual collection is embedded and
30 referenced through a TSeqCollection base pointer. The class of the embedded object for any given
31 instance is passed as an argument to the constructor:
32 <pre>
33 <code class=C++>
34 KVSeqCollection* my_coll = new KVSeqCollection("THashList");
35 </code>
36 </pre>
37 Any collection class derived from TSeqCollection is valid: this means all ordered collections, for which the order in which
38 objects are added is preserved. Some of these lists can also be sorted (see below)<br>
39 Any object derived from TObject can be stored in the collection, but only objects
40 derived from KVBase can be sought using their 'type', 'label', or 'number' attributes
41 (these characteristics are not defined for TObject). Note that we can test the 'KVBase'-ness of any object through a
42 TObject* base pointer using the 'KaliVeda' bit, KVBase::kIsKaliVedaObject:
43 <pre>
44 <code class=C++>
45 TObject *obj = [address of some object]
46 if( obj->TestBit( KVBase::kIsKaliVedaObject ) ){
47  &frasl;&frasl; object derives from KVBase
48 }
49 </code>
50 </pre>
51 <h3>Signals &amp; slots: monitoring list modification</h3>
52 If SendModifiedSignals(kTRUE) is called, then every time an object is added to or removed from the list, it will emit
53 the signal "Modified()". This can be used to monitor any changes to the
54 state of the list.<br>
55 <h3>FindObjectBy...</h3>
56 In addition to the standard TList::FindObject(const Char_t* name) and
57 TList::FindObject(TObject*) methods, KVSeqCollection adds methods to find
58 objects based on several different properties, such as type, label,
59 number, or class. Note that type, label and number are only defined for
60 objects derived from KVBase; if the list contains a mixture of TObject-
61 and KVBase-derived objects, the TObject-derived objects will be ignored
62 if type, label or number are used to search.<br>
63 Finally, the very general FindObjectWithMethod method can search for an
64 object using any valid method.<br>
65 <h3>Sublists</h3>
66 The GetSubListWith... methods allow to generate new lists containing a
67 subset of the objects in the main list, according to their name, label,
68 type, etc. These sublists do not own their objects (they are supposed
69 to be owned by the main list), and must be deleted by the user in order
70 to avoid memory leaks.<br>
71 <h3>Execute</h3>
72 The Execute methods can be used to execute a given method with the same
73 arguments for every object in the list.<br>
74 <h3>Sorting lists</h3>
75 No 'Sort()' method is defined in base class TSeqCollection, although the TCollection::IsSortable()
76 method returns kTRUE by default (and is not overridden in child classes which do not implement
77 sorting). Therefore whether or not a KVSeqCollection can be sorted depends on the collection
78 class which is being used. We do not define a KVSeqCollection::Sort method. If you want to sort
79 a KVSeqCollection and you know that the embedded collection class has a valid Sort method,
80 you can do as follows:
81 <pre>
82 <code class=C++>
83 KVSeqCollection* seqlist = new KVSeqCollection("TList");
84 seqlist->Add(...);
85 ...
86 ((TList*)seqlist->GetCollection())->Sort();
87 </code>
88 </pre>
89 or perhaps more cautiously you should do:
90 <pre>
91 <code class=C++>
92 TList* tlist = dynamic_cast&lt;TList*&gt;(seqlist->GetCollection());
93 if(tlist) tlist->Sort();
94 </code>
95 </pre>

328 void KVSeqCollection::Copy(TObject& obj) const
329 {
330  // Copy a list of objects, including the name of the list.
331  // If this list owns its objects, we make new Clones of all objects in the list
332  // (N.B. the Clone() method must work correctly for the objects in question)
333  // and put them in the copy list, the copy will own these new objects.
334  // Copy will have same IsOwner and IsCleanup status as this list.
335  // If this list sends Modified() signal, the copy will do too.
336 
337  TSeqCollection::Copy(obj); //in fact this calls TObject::Copy, no Copy method defined for collection classes
338  KVSeqCollection& copy = (KVSeqCollection&) obj;
339  copy.SetName(GetName());
340 
341  //clear any pre-existing objects in copy list
342  if (copy.IsOwner()) copy.Delete();
343  else copy.Clear();
344 
345  //set ownership
346  copy.SetOwner(IsOwner());
347  //set cleanup status
348  copy.SetCleanup(IsCleanup());
349  //set signal&slot status
350  copy.SendModifiedSignals(IsSendingModifiedSignals());
351 
352  //copy or clone list members
353  TObject* b;
354  TIter next(fCollection);
355  while ((b = next())) {
356  if (IsOwner())
357  copy.Add(b->Clone());
358  else
359  copy.Add(b);
360  }
361 }

680 TObject* KVSeqCollection::FindObjectWithMethod(const Char_t* retvalue, const Char_t* method) const
681 {
682  // Find the first object in the list for which the given method returns the given return value:
683  // e.g. if method = "GetName" and retvalue = "john", we return the
684  // first object in this list for which GetName() returns "john".
685  //
686  // For each object of the list, the existence of the given method is checked using TMethodCall::IsValid()
687  // if the method is valid and the return value is equal to the input one (retvalue) object is returned
688  // Supported return types are those defined in TMethodCall::ReturnType()
689 
690  if (retvalue && method) {
691  KVString RV(retvalue);
692  KVString MTH(method);
693  Bool_t wildcard = RV.Contains("*");
694  TIter next(fCollection);
695  TObject* obj;
696  while ((obj = next())) {
697  TMethodCall mt;
698  mt.InitWithPrototype(obj->IsA(), MTH.Data(), "");
699  if (mt.IsValid()) {
700  if (mt.ReturnType() == TMethodCall::kString) {
701  Char_t* ret;
702  mt.Execute(obj, "", &ret);
703  if (ret != nullptr) {
704  if (!wildcard) {
705  if (RV == ret) {
706  return obj;
707  }
708  } else {
709  if (KVString(ret).Match(RV)) {
710  return obj;
711  }
712  }
713  }
714  } else if (mt.ReturnType() == TMethodCall::kLong) {
715  Long_t ret;
716  mt.Execute(obj, "", ret);
717  if (ret == RV.Atoi()) {
718  return obj;
719  }
720  } else if (mt.ReturnType() == TMethodCall::kDouble) {
721  Double_t ret;
722  mt.Execute(obj, "", ret);
723  if (ret == RV.Atof()) {
724  return obj;
725  }
726  } else Error("FindObjectWithMethod", "Return type %d is not supported", (int)mt.ReturnType());
727  }
728  }
729  }
730  return nullptr;
731 
732 }

793 TObject* KVSeqCollection::FindObjectAny(const Char_t* att, const Char_t* keys, Bool_t contains_all, Bool_t case_sensitive) const
794 {
795  //Find an object in the list, if one of its characteristics 'att' contains any or all of
796  //the keywords contained in the string 'keys'
797  //
798  // att = "name", "title", "class", "type", "label",
799  // WARNING: when using "type" or "label", any objects in list which do
800  // not inherit from KVBase are ignored
801  // keys = list of keywords, separated by spaces
802  //
803  // contains_all = kFALSE (default) : object found if any keyword occurs in the characteristic 'att'
804  // contains_all = kTRUE : object found if all keywords occur in the characteristic 'att'
805  //
806  // case_sensitive = kTRUE (default) : case-sensitive comparison
807  // case_sensitive = kFALSE : ignore case of keywords
808 
809  int char_test = -1;
810  enum { kName, kTitle, kClass, kType, kLabel };
811  if (!strcmp(att, "name")) char_test = kName;
812  else if (!strcmp(att, "title")) char_test = kTitle;
813  else if (!strcmp(att, "type")) char_test = kType;
814  else if (!strcmp(att, "label")) char_test = kLabel;
815  else if (!strcmp(att, "class")) char_test = kClass;
816  else return nullptr;
817 
818  TString::ECaseCompare casecmp;
819  if (case_sensitive) casecmp = TString::kExact;
820  else casecmp = TString::kIgnoreCase;
821 
822  TString _keys(keys);
823  unique_ptr<TObjArray> keywords(_keys.Tokenize(' '));
824  if (!keywords.get()) return nullptr;
825  int nkeys;
826  if (!(nkeys = keywords->GetEntries())) {
827  return nullptr;
828  }
829 
830  int nmatches;
831  TIter next(fCollection);
832  TString _att;
833  TObject* obj(nullptr);
834  KVBase* kvobj(nullptr);
835  while ((obj = next())) {
836 
837  if (char_test > kClass && !obj->TestBit(KVBase::kIsKaliVedaObject)) {
838  continue;
839  }
840  switch (char_test) {
841  case kName:
842  _att = obj->GetName();
843  break;
844  case kTitle:
845  _att = obj->GetTitle();
846  break;
847  case kClass:
848  _att = obj->ClassName();
849  break;
850  case kLabel:
851  _att = kvobj->GetLabel();
852  break;
853  case kType:
854  _att = kvobj->GetType();
855  break;
856  }
857  nmatches = 0;
858  for (int i = 0; i < nkeys; i++) {
859  nmatches += (_att.Contains(((TObjString*)keywords->At(i))->String(), casecmp));
860  }
861  if ((nmatches && !contains_all) || ((nmatches == nkeys) && contains_all)) {
862  return obj;
863  }
864  }
865  return nullptr;
866 }

958  // Recursively create and fill a (sub)list with objects in this list (and any sublists) for which the
959  // given method returns the given return value:

961  // (sub)list of objects in this list for which GetName() returns "john".
962  //
963  // This new list will be of the same kind as this one.
964  // The objects in the sublist do not belong to the sublist.

967  // unique_ptr<KVSeqCollection> ptr(GetSubListWithMethod(...));
968  //
969  // For each object of the list, the existence of the given method is checked using TMethodCall::IsValid()
970  // if the method is valid and the return value is equal to the input one (retvalue) object is added to the subKVList
971  // return type supported are those defined in TMethodCall::ReturnType()
972 
973  KVSeqCollection* sublist = NewCollectionLikeThisOne();

983 void KVSeqCollection::_GetSubListWithMethod(KVSeqCollection* outputList, TCollection* Col, const Char_t* retvalue, const Char_t* method) const
984 {
985  if (retvalue && method) {
986  KVString RV(retvalue);
987  KVString MTH(method);
988  Bool_t wildcard = RV.Contains("*");
989  TIter next(Col);
990  TObject* ob;
991  while ((ob = next())) {
992  // recursive search in subidrectories
993  if (ob->InheritsFrom("TCollection")) {
994  _GetSubListWithMethod(outputList, (TCollection*)ob, retvalue, method);
995  continue;
996  }
997  TMethodCall mt;
998  mt.InitWithPrototype(ob->IsA(), MTH.Data(), "");
999  if (mt.IsValid()) {
1000  //cout << "it is valid" << endl;
1001  if (mt.ReturnType() == TMethodCall::kString) {
1002  Char_t* ret;
1003  mt.Execute(ob, "", &ret);
1004  if (!wildcard) {
1005  if (RV == ret) outputList->Add(ob);
1006  } else {
1007  if (KVString(ret).Match(RV)) outputList->Add(ob);
1008  }
1009  } else if (mt.ReturnType() == TMethodCall::kLong) {
1010  Long_t ret;
1011  mt.Execute(ob, "", ret);
1012  if (ret == RV.Atoi()) outputList->Add(ob);
1013  } else if (mt.ReturnType() == TMethodCall::kDouble) {
1014  Double_t ret;
1015  mt.Execute(ob, "", ret);
1016  if (ret == RV.Atof()) outputList->Add(ob);
1017  } else std::cout << "this type is not supported " << (int)mt.ReturnType() << std::endl;
1018  }
1019  }
1020  }
