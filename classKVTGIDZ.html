Create Z identification with given "name", using the generalised Tassan-Got functional KVTGIDFunctions::fede. npar = total number of parameters type = functional type (0: standard, 1:extended) light = with (1) or without (0) CsI light-energy dependence mass = mass formula used to calculate A for each Z (see KVNucleus::GetAFromZ)
