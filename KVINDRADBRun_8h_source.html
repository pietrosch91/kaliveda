67  //The GENE DIRECT of INDRA Selector module is generated from an internal clock with a nominal
68  //frequency of 160Hz, but which in reality (as measured by scaler) is approximately 185Hz (it actually
69  //appears to fluctuate around this value). Therefore the length of a run can be obtained
70  //from the GD simply by dividing by 185 (or another value for the average frequency of the pulser,
71  //which may be given as argument). This method returns the result of this
72  //division, corresponding to the length of the run (in seconds).
73  virtual Double_t GetGeneDirectTime(Double_t average_gene_dir_frequency = 185.0) const
