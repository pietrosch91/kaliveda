The Minimiser module

MEDetectorStack.cpp

Description

A simple Silicon-Isobutane-CsI detector stack, used in the A value estimation algorithms - see SiliconEnergyMinimiser. Allows one to calculate the "delta" between the measured and simulated silicon detector energies in order to determine the most likely candidate for the A value (ideally "delta" would be zero so we look for the minimum value of "delta").
