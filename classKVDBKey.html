This function adds a record to the list of cross-references if fIsUnique is kTRUE we check if there is already an object with the same name: if so we return kFALSE otherwise the record is added and kTRUE is returned. if fSingle is kTRUE the list can handle only one object: if the list is empty the record is added and kTRUE is returned, otherwise kFALSE. check if more than one object is allowed
