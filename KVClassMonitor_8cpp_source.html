14 <h2>KVClassMonitor</h2>
15 <h4>track potential memory leaks</h4>
16 
17 Simple tool to track which classes instance count increases over time.
18 Every time the Check() method is called we compare the instance count of every
19 class derived from TObject to the previous Check().
20 We print a warning message for every class whose instance count has increased.
21 
22 YOU MUST ACTIVATE ROOT OBJECT TRACKING FOR THIS TO WORK!
23 
24 In your .rootrc file, add/change this variable:
25 
26  Root.ObjectStat: 1
27 
28 Note that, as the instance counts are stored in a TEnv, any time new classes are
29 instanciated, some class instances increase due to the TEnv itself changing.
30 Typical output in this case is:
31 
32  Info in <KVClassMonitor::Check>: Checking class instance statistics
33  Warning in <KVClassMonitor::Check>: TList increase from 4060 --> 4065
34  Warning in <KVClassMonitor::Check>: TEnvRec increase from 3032 --> 3038
35  Warning in <KVClassMonitor::Check>: TClass increase from 70 --> 71
36  Warning in <KVClassMonitor::Check>: TObjArray increase from 90 --> 92
37 
38 ### Usage
39 This is a singleton class: only one instance can exist at a time. A pointer to
40 the current instance can be retrieved by static method KVClassMonitor::GetInstance().
41 Typical use:
42 
43  KVClassMonitor* cmon = KVClassMonitor::GetInstance();
44 
45  while( [some treatment loop] ){
46 
47  [lots of memory-leaking activity]
48 
49  cmon->Check();// print infos on memory usage
50  }

178 void KVClassMonitor::Check()
179 {
180  // update class instance statistics
181  // print warning for every class whose number has increased
182  Info("Check", "Checking class instance statistics");
183  gObjectTable->UpdateInstCount();
184  TIter next(gROOT->GetListOfClasses());
185  TClass* cl;
186  while ((cl = (TClass*) next())) {
187  Int_t n = cl->GetInstanceCount();
188  Int_t old = fClassStats.GetValue(cl->GetName(), 0);
189  if (n != old) {
190  if (n > old) Warning("Check", "%s +%d (%d --> %d)", cl->GetName(), n - old, old, n);
191  else Warning("Check", "%s -%d (%d --> %d)", cl->GetName(), old - n, old, n);
192  }
193  fClassStats.SetValue(cl->GetName(), n);
194  }
195 }
