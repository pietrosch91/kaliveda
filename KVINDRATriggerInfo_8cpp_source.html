23 ClassImp(KVINDRATriggerInfo)
25 //Utility class, holds values of the STAT_EVE, R_DEC and CONFIG parameters
26 //in raw INDRA data which represent the status of the Selecteur for an event.
27 //
28 //When reading raw data, the values of the three parameters
29 //are filled from the data event by event.
30 //For calibrated data, this information is accessible by using
31 // KVINDRAReconEvent::EventTrigger()
32 //which returns a KVTriggerInfo object.
33 //
34 //GetSTAT_EVE(), GetR_DEC(), GetCONFIG() : binary contents of each register
35 //PHY_EVT(), MRQ(), GEN_ELEC(), GEN_TST(), GEN_LAS() : kTRUE/kFALSE depending on state of corresponding bit in STAT_EVE
36 //IsGene() : kTRUE if any of the three GEN_xxx bits is set to 1
37 //IsPhysics() : kTRUE if PHY_EVT bit = 1 and MRQ bit is not set*
38 //(*because if MRQ=1 physical events cannot trigger the acquisition).
