KVedaLoss Class Reference
The Stopping module

C++ implementation of VEDALOSS stopping power calculation.

See documentation here.

Energy limits Normally all range, dE, Eres functions are limited to range 0<=E<=Emax, where Emax is nominal max energy for which range tables are valid (usually 400MeV/u for Z<3, 250MeV/u for Z>3) If higher energies are required, call static method KVedaLoss::SetIgnoreEnergyLimits() BEFORE ANY MATERIALS ARE CREATED in order to recalculate the Emax limits in such a way that: range function is always monotonically increasing function of Einc stopping power is concave (i.e. no minimum of stopping power followed by an increase) at the most, the new limit will be 1 GeV/nucleon. at the least, it will remain at the nominal (400 or 250 MeV/nucleon) level.
