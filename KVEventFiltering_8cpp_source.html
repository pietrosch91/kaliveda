20 <h2>KVEventFiltering</h2>
21 <h4>Filter simulated events with multidetector response</h4>
22 
23 Use this KVEventSelector on simulated data TTrees containing a branch with KVSimEvent-derived objects:
24 
25  my_tree->Process("KVEventFiltering", "[options]");
26 
27 where `"[options]"` is the list of options in the form `"BranchName=toto,Dataset=titi,System=tata, ..."`.
28 
29 The following options __must__ be given:
30 
31  - `SimFileName`= name of file containing simulated events
32  - `SimTitle`= description of simulation
33  - `BranchName`= name of branch containing simulated events
34  - `Dataset`= name of experimental dataset (defines multidetector array to use etc.)
35  - `System`= name of experimental dataset system (defines collision kinematics etc.)
36  - `Geometry`= type of geometry, either 'KV' (KaliVeda geometry) or 'ROOT' (ROOT TGeometry package)
37  - `Filter`= type of filter, either `'Geo'` (geometric filter), `'GeoThresh'` (geometry + detector thresholds),
38  or `'Full'` (full simulation of detector response, including experimental identification
39  and calibration routines)
40  - `OutputDir`= directory path in which to write filtered data file
41  - `Kinematics`= kinematical frame for simulation, either `"cm"` or `"lab"`. if `"cm"`, we use the c.m. velocity
42  of the selected `System` to transform events into the detector (laboratory) frame.
43  if `"lab"` no transformation is performed: simulated events are already in laboratory frame.
44 
45 The following are optional options:
46 
47  - `Run`: run number to use for detector status, setup, parameters, etc.
48  if not given, first run of the given experimental system is used.
49  - `PhiRot`: by default, a random rotation around the beam axis will be performed before
50  simulating detection of the event. If you don't want this to happen,
51  give option PhiRot=no
52  If used, filtered events will have a parameter "RANDOM_PHI" with the applied
53  rotation (in radians)
54  - `Gemini`: if option Gemini=yes, then each event will be "decayed" with Gemini++,
55  if KaliVeda has been compiled with Gemini++ support. See below for extra
56  information on Gemini decay stored in particle parameter lists.
57  - `GemDecayPerEvent`: if option Gemini=yes then by default 1 Gemini++ decay will be performed for each event.
58  you can change this by giving a value for this option
59 
60 The filtered data will be written in the directory given as option "OutputDir".
61 The filename is built up from the original simulation filename and the values
62 of various options:
63 
64  [simfile]_geo=[geometry]_filt=[filter-type]_[dataset]_[system]_run=[run-number].root
65 
66 The data will be stored in a TTree with name 'ReconstructedEvents', in a branch with name
67 'ReconEvent'. The class used for reconstructed events depends on the dataset,
68 it is given by KVDataSet::GetReconstructedEventClassName().
69 
70 Each filtered event will have some or all of the following parameters in its list:
71 
72  - SIMEVENT_TREE_ENTRY = index of simulated event in TTree we are reading
73  - SIMEVENT_NUMBER = event number of simulated event in TTree we are reading (if defined i.e. if !=0)
74  - RANDOM_PHI = rotation around beam axis [in radians]
75 
76 When using Gemini++ to decay simulated events before filtering, each particle will have
77 the following parameter defined:
78 
79  - GEMINI_PARENT_INDEX = index of parent nucleus in simulated event
80 
81 The combination of SIMEVENT_TREE_ENTRY/_NUMBER and SIMEVENT_NUC allows to connect primary
82 fragments in the simulated events with their detected decay products in the filtered event.
83 
84 Example: suppose 'recev' is a filtered event and 'sim_tree' is the TTree containing the original
85 simulation, while 'simev' is a pointer to a KVSimEvent object connected to the appropriate
86 branch in 'sim_tree'. Then we can retrieve the parent nucleus responsible for producing
87 a particular nucleus in 'recev' like so:
88 
89  sim_tree.GetEntry( recev.GetIntValue("SIMEVENT_TREE_ENTRY" ) );
90  KVSimNucleus* parent_nuc = (KVSimNucleus*)simev->GetParticle( recev.GetNucleus(15)->GetIntValue("GEMINI_PARENT_INDEX") ); // parent nucleus of 15th nucleus in filtered event 'recev'
91 
92 
93 */

264 void KVEventFiltering::RandomRotation(KVEvent* to_rotate, const TString& frame_name) const
265 {
266  // do random phi rotation around z-axis
267  // if frame_name is given, apply rotation to that frame
268  //
269  // store phi rotation angle [radians] in event parameter "RANDOM_PHI"
270  TRotation r;
271  Double_t phi = gRandom->Uniform(TMath::TwoPi());
272  r.RotateZ(phi);
273  if (frame_name != "") to_rotate->SetFrame("rotated_frame", frame_name, r);
274  else to_rotate->SetFrame("rotated_frame", r);
275  to_rotate->SetParameter("RANDOM_PHI", phi);

288 Bool_t KVEventFiltering::Analysis()
289 {
290  // Event-by-event filtering of simulated data.
291  // If needed (fTransformKinematics = kTRUE), kinematics of event are transformed
292  // to laboratory frame using C.M. velocity calculated in InitAnalysis().
293  // Detection of particles in event is simulated with KVMultiDetArray::DetectEvent,
294  // then the reconstructed detected event is treated by the same identification and calibration
295  // procedures as for experimental data.
296 
297 #ifdef DEBUG_FILTER
298  if (fEVN == 2500) memory_check->SetInitStatistics();
299  else if (fEVN > 4995) memory_check->CompareToInit();
300 #endif
301  KVEvent* to_be_detected = GetEvent();
302  Int_t iterations = 1;
303  if (fGemini) iterations = fGemDecayPerEvent;
304  if (to_be_detected->GetNumber()) to_be_detected->SetParameter("SIMEVENT_NUMBER", (int)to_be_detected->GetNumber());
305  to_be_detected->SetParameter("SIMEVENT_TREE_ENTRY", (int)fTreeEntry);
306 #ifdef WITH_GEMINI
307  do {
308  KVGemini g;
309  try {
310  g.DecayEvent((KVSimEvent*)GetEvent(), &fGemEvent);
311  }
312  catch (...) {
313  continue;
314  }
315  to_be_detected = &fGemEvent;
316  //Copy any parameters associated with simulated event into the Gemini-decayed event
317  GetEvent()->GetParameters()->Copy(*(fGemEvent.GetParameters()));
318 #endif
319  if (fTransformKinematics) {
320  if (fNewFrame == "proj") to_be_detected->SetFrame("lab", fProjVelocity);
321  else to_be_detected->SetFrame("lab", fCMVelocity);
322  if (fRotate) {
323  RandomRotation(to_be_detected, "lab");
324  gMultiDetArray->DetectEvent(to_be_detected, fReconEvent, "rotated_frame");
325  }
326  else {
327  gMultiDetArray->DetectEvent(to_be_detected, fReconEvent, "lab");
328  }
329  }
330  else {
331  if (fRotate) {
332  RandomRotation(to_be_detected);
333  gMultiDetArray->DetectEvent(to_be_detected, fReconEvent, "rotated_frame");
334  }
335  else {
336  gMultiDetArray->DetectEvent(to_be_detected, fReconEvent);
337  }
338  }
339  fReconEvent->SetNumber(fEVN++);
340  fReconEvent->SetFrameName("lab");
341  fTree->Fill();
342 #ifdef WITH_GEMINI
343  }
344  while (fGemini && --iterations);
345 #endif
346 
347  return kTRUE;
348 }

384 void KVEventFiltering::InitAnalysis()
385 {
386  // Select required dataset for filtering (option "Dataset")
387  // Build the associated multidetector geometry.
388  // Calculate C.M. velocity associated with required experimental collision
389  // kinematics (option "System").
390  //
391  // Set the parameters of the detectors according to the required run
392  // if given (option "Run"), or the first run of the given system otherwise.
393  // If ROOT/TGeo geometry is required (option "Geometry"="ROOT"),
394  // build the TGeometry representation of the detector array.
395  //
396  // Open file for filtered data (see KVEventFiltering::OpenOutputFile), which will
397  // be stored in a TTree with name 'ReconstructedEvents', in a branch with name
398  // 'ReconEvent'. The class used for reconstructed events depends on the dataset,
399  // it is given by KVDataSet::GetReconstructedEventClassName().
400 
401  if (fDisableCreateTreeFile) return; //when running with PROOF, only execute on workers
402 
403  TString dataset = GetOpt("Dataset").Data();
404  if (!gDataSetManager) {
405  gDataSetManager = new KVDataSetManager;
406  gDataSetManager->Init();
407  }
408  gDataSetManager->GetDataSet(dataset)->cd();
409 
410  TString system = GetOpt("System").Data();
411  KVDBSystem* sys = (gDataBase ?
412  (gDataBase->GetTable("Systems") ? (KVDBSystem*)gDataBase->GetTable("Systems")->GetRecord(system) : nullptr)
413  : nullptr);
414  KV2Body* tb = 0;
415 
416  Bool_t justcreated = kFALSE;
417  if (sys) tb = sys->GetKinematics();
418  else if (system) {
419  tb = new KV2Body(system);
420  tb->CalculateKinematics();
421  justcreated = kTRUE;
422  }
423 
424  fCMVelocity = (tb ? tb->GetCMVelocity() : TVector3(0, 0, 0));
425  fCMVelocity *= -1.0;
426 
427  fProjVelocity = (tb ? tb->GetNucleus(1)->GetVelocity() : TVector3(0, 0, 0));
428  fProjVelocity *= -1.0;
429 
430  if (justcreated)
431  delete tb;
432 
433  Int_t run = 0;
434  if (IsOptGiven("Run")) {
435  run = GetOpt("Run").Atoi();
436  Info("InitAnalysis", "Run given in options = %d", run);
437  }
438  if (!run) {
439  if (sys) {
440  run = ((KVDBRun*)sys->GetRuns()->First())->GetNumber();
441  Info("InitAnalysis", "Using first run for system = %d", run);
442  }
443  else {
444  Info("InitAnalysis", "No run information");
445  run = -1;
446  }
447  }
448 
449  KVMultiDetArray::MakeMultiDetector(dataset, run);
450  if (run == -1) gMultiDetArray->InitializeIDTelescopes();
451  gMultiDetArray->SetSimMode();
452 
453  TString geo = GetOpt("Geometry").Data();
454  if (geo == "ROOT") {
455  gMultiDetArray->CheckROOTGeometry();
456  Info("InitAnalysis", "Filtering with ROOT geometry");
457  Info("InitAnalysis", "Navigator detector name format = %s", gMultiDetArray->GetNavigator()->GetDetectorNameFormat());
458  }
459  else {
460  gMultiDetArray->SetROOTGeometry(kFALSE);
461  Info("InitAnalysis", "Filtering with KaliVeda geometry");
462  }
463 
464  TString filt = GetOpt("Filter").Data();
465  if (filt == "Geo") {
466  gMultiDetArray->SetFilterType(KVMultiDetArray::kFilterType_Geo);
467  Info("InitAnalysis", "Geometric filter");
468  }
469  else if (filt == "GeoThresh") {
470  gMultiDetArray->SetFilterType(KVMultiDetArray::kFilterType_GeoThresh);
471  Info("InitAnalysis", "Geometry + thresholds filter");
472  }
473  else if (filt == "Full") {
474  gMultiDetArray->SetFilterType(KVMultiDetArray::kFilterType_Full);
475  Info("InitAnalysis", "Full simulation of detector response & calibration");
476  }
477  TString kine = GetOpt("Kinematics").Data();
478  if (kine == "lab") {
479  fTransformKinematics = kFALSE;
480  Info("InitAnalysis", "Simulation is in laboratory/detector frame");
481  }
482  else {
483  fNewFrame = kine;
484  Info("InitAnalysis", "Simulation will be transformed to laboratory/detector frame");
485  }
486 
487  if (IsOptGiven("PhiRot")) {
488  if (GetOpt("PhiRot") == "no") fRotate = kFALSE;
489  }
490  if (fRotate) Info("InitAnalysis", "Random phi rotation around beam axis performed for each event");
491 #ifdef WITH_GEMINI
492  if (IsOptGiven("Gemini")) {
493  if (GetOpt("Gemini") == "yes") fGemini = kTRUE;
494  if (IsOptGiven("GemDecayPerEvent")) fGemDecayPerEvent = GetOpt("GemDecayPerEvent").Atoi();
495  else fGemDecayPerEvent = 1;
496  }
497  if (fGemini) Info("InitAnalysis", "Statistical decay with Gemini++ for each event before detection");
498  if (fGemDecayPerEvent > 1) Info("InitAnalysis", " -- %d decays per primary event", fGemDecayPerEvent);
499 #endif
500  if (!gDataSet->HasCalibIdentInfos()) {
501  TString fullpath;
502  if (KVBase::SearchKVFile("IdentificationBilan.dat", fullpath, gDataSet->GetName())) {
503  Info("InitAnalysis", "Setting identification bilan...");
504  TString sysname = sys->GetBatchName();
505  KVIDTelescope::OpenIdentificationBilan(fullpath);
506  TIter next(gMultiDetArray->GetListOfIDTelescopes());
507  KVIDTelescope* idt;
508  while ((idt = (KVIDTelescope*)next())) {
509  idt->CheckIdentificationBilan(sysname);
510  }
511  }
512  }
513  gMultiDetArray->PrintStatusOfIDTelescopes();
514 
515  OpenOutputFile(sys, run);
516  if (sys) fTree = new TTree("ReconstructedEvents", Form("%s filtered with %s (%s)", GetOpt("SimTitle").Data(), gMultiDetArray->GetTitle(), sys->GetName()));
517  else fTree = new TTree("ReconstructedEvents", Form("%s filtered with %s", GetOpt("SimTitle").Data(), gMultiDetArray->GetTitle()));
518 
519  TString reconevclass = gDataSet->GetReconstructedEventClassName();
520  fReconEvent = (KVReconstructedEvent*)TClass::GetClass(reconevclass)->New();
521  KVEvent::MakeEventBranch(fTree, "ReconEvent", reconevclass, &fReconEvent);
522 
523  AddTree(fTree);
524 }

562 void KVEventFiltering::OpenOutputFile(KVDBSystem* S, Int_t run)
563 {
564  // Open ROOT file for new filtered events TTree.
565  // The file will be written in the directory given as option "OutputDir".
566  // The filename is built up from the original simulation filename and the values
567  // of various options:
568  //
569  // [simfile]_[Gemini]_geo=[geometry]_filt=[filter-type]_[dataset]_[system]_run=[run-number].root
570  //
571  // In addition, informations on the filtered data are stored in the file as
572  // TNamed objects. These can be read by KVSimDir::AnalyseFile:
573  //
574  // KEY: TNamed System;1 title=[full system name]
575  // KEY: TNamed Dataset;1 title=[dataset name]
576  // KEY: TNamed Run;1 title=[run-number]
577  // KEY: TNamed Geometry;1 title=[geometry-type]
578  // KEY: TNamed Filter;1 title=[filter-type]
579  // KEY: TNamed Origin;1 title=[name of simulation file]
580  // KEY: TNamed RandomPhi;1 title=[yes/no, random rotation about beam axis]
581  // KEY: TNamed Gemini++;1 title=[yes/no, Gemini++ decay before detection]
582  // KEY: TNamed GemDecayPerEvent;1 title=[number of Gemini++ decays per primary event]
583  //
584  TString basefile = GetOpt("SimFileName");
585  basefile.Remove(basefile.Index(".root"), 5);
586  TString outfile = basefile;
587  if (fGemini) {
588  outfile += "_Gemini";
589  if (fGemDecayPerEvent > 1) outfile += fGemDecayPerEvent;
590  }
591  outfile += "_geo=";
592  outfile += GetOpt("Geometry");
593  outfile += "_filt=";
594  outfile += GetOpt("Filter");
595  outfile += "_";
596  outfile += gDataSet->GetName();
597 
598  if (S) {
599  outfile += "_";
600  outfile += S->GetBatchName();
601  outfile += "_run=";
602  outfile += Form("%d", run);
603  }
604  else if (GetOpt("System")) {
605  TString tmp = GetOpt("System");
606  tmp.ReplaceAll(" ", "");
607  tmp.ReplaceAll("/", "");
608  tmp.ReplaceAll("@", "");
609  tmp.ReplaceAll("MeV", "");
610  tmp.ReplaceAll("A", "");
611  tmp.ReplaceAll("+", "");
612  outfile += "_";
613  outfile += tmp.Data();
614  outfile += "_run=0";
615  }
616  outfile += ".root";
617 
618  TString fullpath;
619  AssignAndDelete(fullpath, gSystem->ConcatFileName(GetOpt("OutputDir").Data(), outfile.Data()));
620 
621  //fFile = new TFile(fullpath,"recreate");
622  CreateTreeFile(fullpath);
623 
624  TDirectory* curdir = gDirectory;
625  writeFile->cd();
626  if (S) {
627  TNamed* system = new TNamed("System", S->GetName());
628  system->Write();
629  }
630  else if (GetOpt("System"))(new TNamed("System", GetOpt("System").Data()))->Write();
631 
632  (new TNamed("Dataset", gDataSet->GetName()))->Write();
633  if (S)(new TNamed("Run", Form("%d", run)))->Write();
634  if (gMultiDetArray->IsROOTGeometry()) {
635  (new TNamed("Geometry", "ROOT"))->Write();
636  }
637  else {
638  (new TNamed("Geometry", "KV"))->Write();
639  }
640  (new TNamed("Filter", GetOpt("Filter").Data()))->Write();
641  (new TNamed("Origin", (basefile + ".root").Data()))->Write();
642  (new TNamed("RandomPhi", (fRotate ? "yes" : "no")))->Write();
643  (new TNamed("Gemini++", (fGemini ? "yes" : "no")))->Write();
644  (new TNamed("GemDecayPerEvent", Form("%d", fGemDecayPerEvent)))->Write();
645  curdir->cd();
646 }
