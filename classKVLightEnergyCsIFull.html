◆ Compute()

Double_t KVLightEnergyCsIFull::Compute ( Double_t  light) const
virtual

Calculate the calibrated energy (in MeV) for a given total light output. The Z and A of the particle should be given first using SetZ, SetA. By default, Z=A=1 (proton).

This is done by inversion of the light-energy function using TF1::GetX. set parameters of light-energy function

Implements KVCalibrator.
