◆ CalculateMarkerWidths()

void KVIDQALine::CalculateMarkerWidths ( )
virtual

Calculate natural "width" of each ID markers in the line. The markers in the line have first to sorted so that they are in order of ascending 'X' i.e. first markers has lower A, last line is the heaviest mass .

The width is supposed to be the distance between the first and the last marker divided by the difference of A of both markers.
