50  {

52  // The actual method called depends on the value of fROOTGeometry:
53  // fROOTGeometry=kTRUE: calls DetectParticle_TGEO, particle propagation performed using
54  // TGeo description of array and algorithms from ROOT TGeo package
55  // fROOTGeometry=kFALSE: calls DetectParticle_KV, uses simple KaliVeda geometry
56  // to simulate propagation of particle
57  //
58  // The default value is given in .kvrootrc by variable KVASMultiDetArray.FilterUsesROOTGeometry
59  return (IsROOTGeometry() ? DetectParticle_TGEO(part) : DetectParticle_KV(part));
