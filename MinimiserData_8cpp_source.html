11 // MinimiserData.cpp

13 // Description
14 //
15 // Base class for handling data produced by the SiliconEnergyMinimiser. The
16 // minimiser function usually just returns the A value it has determined to be
17 // the most likely for a given Z (i.e. It returns an integer). Therefore to
18 // obtain more interesting information about how well the algorithm is
19 // performing, we need to pass in an (optional) pointer to a data class such as
20 // this one for the minimiser implementation to populate.
21 //
22 // This class handles only the essential data produced by any hypothetical
23 // minimiser implementation. For data specific to the implementation you must
24 // create an inheriting class (See MonoMinimiserData and ThreadedMinimiserData).
25 //
26 // Peter C. Wigg <peter.wigg.314159@gmail.com>
27 // Thu 4 Feb 15:43:34 2016
28 //
