19 /*
20 Cette classe a toute son utilité si on s'attend a avoir dans une liste de partitions
21 une part importante de partition identique un exmeple dans KVBreakUp
22 -----
23 Via la methode Fill(KVIntegerList* )
24 on test la présence ou non d'une partition, si une partition identique est deja
25 presente, on incremente la population de celle-ci KVIntegerList::AddPopulation()
26 sinon on ajoute la partition dans la liste.
27 Le nombre de partition totale est accessible via GetNbreTot()
28 et le nombre de partitions différentes via GetNbreDiff()
29 -----
30 Apres le remplissage, l'utilisateur peut sauvegarder l'ensemble des partitions
31 dans un arbre et l'ecrire dans un fichier root via la methode SaveAsTree
32 */
