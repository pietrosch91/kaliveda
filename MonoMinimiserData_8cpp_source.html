8 // Description
9 //
10 // Class for handling data produced by the SiliconEnergyMinimiser. The minimiser
11 // function usually just returns the A value it has determined to be the most
12 // likely for a given Z (i.e. It returns an integer). Therefore to obtain more
13 // interesting information about how well the algorithm is performing, we need
14 // to pass in an (optional) pointer to a data class such as this one for the
15 // minimiser implementation to populate.
16 //
17 // This class handles the extra data produced by the MonoMinimiserImpl
18 // implementation, in addition to the basic minimiser data (See MinimiserData).
