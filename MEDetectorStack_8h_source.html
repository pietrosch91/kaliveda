10 // A simple Silicon-Isobutane-CsI detector stack, used in the A value estimation
11 // algorithms - see SiliconEnergyMinimiser. Allows one to calculate the "delta"
12 // between the measured and simulated silicon detector energies in order to
13 // determine the most likely candidate for the A value (ideally "delta" would be
