23 <h2>KVBatchSystemManager</h2>
24 <h4>Handles list of all available batch systems for processing non-interactive data analysis tasks</h4>
25 <p>The different "batch" systems available are described in the $KVROOT/KVFiles/.kvrootrc
26 or the user's $HOME/.kvrootrc, as in the following examples:</p>
27 <pre>
28 # Batch systems
29 BatchSystem: Xterm
30 Xterm.BatchSystem.Title: Execute task in an X-terminal window
31 +BatchSystem: Linux
32 Linux.BatchSystem.Title: Execute task in background with output stored in log file
33 #+BatchSystem: BQS
34 #BQS.BatchSystem.Title: Use CCIN2P3 BQS batch system
35 #BQS.BatchSystem.DefaultJobTime: 4000
36 #BQS.BatchSystem.DefaultJobMemory: 256MB
37 #BQS.BatchSystem.DefaultJobDisk: 100MB
38 </pre>
39 <p>For each named batch system, there must be a corresponding plugin which
40 defines the actual class to use:</p>
41 <pre>
42 #Plugins for batch systems
43 Plugin.KVBatchSystem: Xterm KVRootBatch KVMultiDet "KVRootBatch()"
44 +Plugin.KVBatchSystem: Linux KVLinuxBatch KVMultiDet "KVLinuxBatch()"
45 +Plugin.KVBatchSystem: BQS KV_CCIN2P3_BQS KVMultiDet "KV_CCIN2P3_BQS()"
46 </pre>
47 <p>The "default" batch system is defined as follows:</p>
48 <pre>
49 #Default batch system
50 Default.BatchSystem: Xterm
51 </pre>
52 <p>It can be accessed via the global pointer to the batch system manager:</p>
53 <pre>
54 KVBatchSystem* default = gBatchSystemManager->GetDefaultBatchSystem();
55 </pre>

134 void KVBatchSystemManager::Init()
135 {
136  //Set up list of available batch systems
137 
138  gBatchSystem = 0;
139 
140  KVString list = gEnv->GetValue("BatchSystem", "");
141 
142  fBatchSystems.Clear();
143 
144  unique_ptr<TObjArray> systems(list.Tokenize(" "));
145  TIter next(systems.get());
146  TObjString* batch_sys;
147  while ((batch_sys = (TObjString*)next())) {
148 
149  KVBatchSystem* bs = KVBatchSystem::GetBatchSystem(batch_sys->GetString().Data());
150  fBatchSystems.Add(bs);
151  }
152 
153  //set default
154  fDefault = (KVBatchSystem*)fBatchSystems.FindObjectByName(gEnv->GetValue("Default.BatchSystem", ""));
155 }

186 void KVBatchSystemManager::Print(Option_t* opt) const
187 {
188  //Print list of available batch systems
189  // Default is to show numbered list with title of each batch system
190  //if opt="all", print detailed information on each batch system
191  if (!strcmp(opt, "all")) {
