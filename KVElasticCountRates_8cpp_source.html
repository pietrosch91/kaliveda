38 <h2>KVElasticCountRates</h2>
39 <h4>Calculate elastic scattering count rates in multidetector arrays</h4>
40 
41 <p>
42 Use this class to calculate the count rates & energy losses of elastically scattered nuclei
43 in the detectors of a multidetector array. It is assumed that the following
44 global pointers have been initialised before using this class:
45 </p>
46 <ul>
47  <li><pre><a href="KVMultiDetArray.html">gMultiDetArray</a></pre> - points to object describing multidetector array</li>
48  <li><pre><a href="KVDataBase.html">gDataBase</a></pre> - points to database of run-dependent information for experimental dataset</li>
49 </ul>
50 <p>
51 In other words, we assume that the user has chosen a dataset on which she wants
52 to work:
53 </p>
54 <pre><a href="KVDataSetManager.html">gDataSetManager</a>-><a href="KVDataSetManager.html#KVDataSetManager:GetDataSet">GetDataSet</a>("name_of_dataset")-><a href="KVDataSet.html#KVDataSet:cd">cd</a>()</pre>
55 <p>
56 and then initialised the description of the experimental configuration for the dataset:
57 </p>
58 <pre><a href="KVMultiDetArray.html">KVMultiDetArray</a>::<a href="KVMultiDetArray.html#KVMultiDetArray:MakeMultiDetector">MakeMultiDetector</a>(gDataSet->GetName())</pre>
59 <h3>Setting up the calculation</h3>
60 <p>Create a new elastic scattering object defining angular range in which to perform calculation:</p>
61 <pre>KVElasticCountRates ecr(2,45,0,360)</pre>
62 <p>
63 Then call the following methods in any order in order to set up the calculation:</p>
64 <ul>
65  <li><pre>ecr.<a href="#KVElasticCountRates:SetProjectile">SetProjectile</a>(...)</pre></li>
66  <li><pre>ecr.<a href="#KVElasticCountRates:SetRun">SetRun</a>(...)</pre></li>
67 </ul>
68 <p>
69 The SetRun(...) method uses the experimental database in order to determine the target
70 for the run, detector state (gas pressures etc.).
71 </p>
72 <h3>Multilayer targets</h3>
73 <p>
74 When multilayer targets are used, the user can restrict the scattering calculation to one
75 specific component of the target, i.e. the scattering takes place between the projectile
76 and one of the nuclei of the specified layer. Energy losses before and after the scattering
77 in the other layers of the target are of course still taken into account. To set the part of
78 the target where scattering takes place use
79 </p>
80 <pre>ecr.<a href="#KVElasticCountRates:SetTargetScatteringLayer">SetTargetScatteringLayer</a>(...)</pre>
81 <p>
82 with the name of the layer you require (this is normally the name of the element
83 making up the layer - see <a href="KVTarget.html">KVTarget</a> for details).
84 </p>
85 <h3>Inelastic scattering</h3>
86 <p>
87 Calculations can also be performed for inelastic scattering i.e. when the target nucleus is left
88 in an excited state (N.B. we still use the Rutherford elastic scattering cross-section for
89 weighting the energy loss distributions). In order to do this, call
90 </p>
91 <pre>ecr.<a href="#KVElasticCountRates:SetTargetExcitedState">SetTargetExcitedState</a>(...)</pre>
92 <p>
93 with the energy of the excited state of the target after scattering.
94 </p>
95 <h3>Running the calculation, obtaining the results</h3>
96 <p>To perform the calculation, do:</p>
97 <pre>ecr.CaclulateScattering(10000)</pre>
98 <p>Make sure the number of sampled points (default=10000) is large enough for accurate determination of count rates.
99 At the end of calculation, we print infos for all detectors hit by elastic particles, these are the count rates for
100 a nominal beam intensity of 10**7 particles per second, the mean energy loss in the detector,
101 the total integrated cross-section per detector in barn, the number of particles per second per unit area hitting
102 each detector, and the total energy per second per unit area deposited in each detector:</p>
103 <pre>
104 kaliveda [3] esa.CalculateScattering(10000)
105 SI_0101 : N= 204.42/sec. <E>= 890.3 MeV Tot.Xsec=6.687E+01 barn fluence=2.711E+01/sec./cm**2 dissip.=2.413E+04 MeV/sec./cm**2
106 SI_0201 : N= 61.37/sec. <E>= 709.7 MeV Tot.Xsec=2.008E+01 barn fluence=1.413E+01/sec./cm**2 dissip.=1.002E+04 MeV/sec./cm**2
107 SI_0301 : N= 15.29/sec. <E>= 706.4 MeV Tot.Xsec=5.002E+00 barn fluence=2.390E+00/sec./cm**2 dissip.=1.688E+03 MeV/sec./cm**2
108 SI_0401 : N= 6.33/sec. <E>= 699.2 MeV Tot.Xsec=2.070E+00 barn fluence=1.582E+00/sec./cm**2 dissip.=1.106E+03 MeV/sec./cm**2
109 </pre>
110 <p>If you want to see the count rates for a different beam intensity, call</p>
111 <pre>ecr.PrintResults(5.e+06)</pr>
112 <p>For each detector hit we fill three histograms:</p>
113 <pre>
114  OBJ: TH1F SI_0223 Eloss in SI_0223 : 0 at: 0x10898980
115  OBJ: TH1F SI_0223_dW Solid angle of SI_0223 : 0 at: 0x10899540
116  OBJ: TH2F SI_0223_map Map of SI_0223 : 0 at: 0x10e2e7d0
117 </pre>
118 <p>Each histogram can be obtained using</p>
119 <pre>ecr.GetHistos().FindObject("SI_0223_map")</pre>
120 <p>The different informations for each detector can be obtained using</p>
121 <pre>
122 kaliveda [7] ecr.GetDetector("SI_0201").count_rate
123 (double)6.13702995974404857e+01
124 kaliveda [8] ecr.GetDetector("SI_0201").mean_energy
125 (double)7.09659910827807721e+02
126 kaliveda [9] ecr.GetDetector("SI_0201").intXsec
127 (double)2.00758282694385635e+01
128 </pre>
129 <p>
130 See <a href="KVElasticCountRate.html">KVElasticCountRate</a> for details.
131 Warning: the values returned correspond to the last beam intensity value given to PrintResults().
132 </p>

375 void KVElasticCountRates::CalculateScattering(Int_t N)
376 {
377  //Perform scattering 'N' times for current values
378  //of particle Z, A and energy, target excited state, and detector.
379  //Print out for each hit detector the mean energy loss & counting rate
380  //for a beam intensity of 10**7 pps
381 
382  fNtirages = N;
383 
384  fHistos.Delete();
385  fRates.clear();
386 
387  if (!fProj) {
388  cout <<
389  "<KVElasticCountRates::CalculateScattering> : Set projectile properties first"
390  << endl;
391  return;
392  }
393  if (!fEnergy) {
394  cout <<
395  "<KVElasticCountRates::CalculateScattering> : Set projectile energy first"
396  << endl;
397  return;
398  }
399  if (!fTarget) {
400  cout <<
401  "<KVElasticCountRates::CalculateScattering> : No target set. Set run first."
402  << endl;
403  return;
404  }
405 
406  /* -------------------------------------------------------------------------------------------------------------------------- */
407 
408  //set up histograms
409 
410  /* -------------------------------------------------------------------------------------------------------------------------- */
411 
412  // -- histograms for debugging: depth in target and angular distribution
413  if (fDepth)
414  delete fDepth;
415  if (fTheta)
416  delete fTheta;
417  fDepth =
418  new TH1F("hDepth", "Depth (mg/cm2)", 500, 0., fTarget->GetTotalEffectiveThickness());
419  fTheta = new TH1F("hTheta", "Theta (deg.)", 500, 0., 0.);
420 
421  /* -------------------------------------------------------------------------------------------------------------------------- */
422 
423  //set up kinematics
424  if (!fKinematics)
425  fKinematics = new KV2Body;
426  fProj->SetEnergy(fEnergy);
427  fProj->SetTheta(0);
428 
429  /* -------------------------------------------------------------------------------------------------------------------------- */
430 
431  //set random interaction point for scattering
432  if (fIntLayer) {
433  fTarget->SetInteractionLayer(fIntLayer, fBeamDirection);
434  } else {
435  fTarget->GetInteractionPoint(fProj);
436  }
437 
438  /* -------------------------------------------------------------------------------------------------------------------------- */
439 
440  //get target nucleus properties from scattering layer
441  TVector3 IP = fTarget->GetInteractionPoint();
442  KVMaterial* targ_mat = fTarget->GetLayer(IP);
443  if (!fTarg) fTarg = new KVNucleus;
444  fTarg->SetZ((Int_t) targ_mat->GetZ());
445  fTarg->SetA((Int_t) targ_mat->GetMass());
446  fKinematics->SetTarget(fTarg);
447 
448  /* -------------------------------------------------------------------------------------------------------------------------- */
449 
450  //set excited state of target nucleus - in other words, dissipated energy for
451  //reaction due to excitation of target
452  fKinematics->SetEDiss(fExx);
453 
454  /* -------------------------------------------------------------------------------------------------------------------------- */
455 
456  for (int i = 0; i < N; i++) {
457  //calculate slowing of incoming projectile
458  fTarget->SetIncoming();
459  fTarget->DetectParticle(fProj);
460  fKinematics->SetProjectile(fProj);
461  fKinematics->SetOutgoing(fProj);
462  fKinematics->CalculateKinematics();
463  //set random direction of outgoing projectile
464  fAngularRange.GetRandomAngles(theta, phi, "random");
465  xsec = TMath::Abs(fKinematics->GetXSecRuthLab(theta));
466  //set energy of scattered nucleus
467  //WARNING: for inverse kinematics reactions, their are two energies for
468  //each angle below the maximum scattering angle.
469  //We only use the highest energy corresponding to the most forward CM angle.
470  Double_t e1, e2;
471  fKinematics->GetELab(3, theta, 3, e1, e2);
472  fProj->SetEnergy(TMath::Max(e1, e2));
473  fProj->SetTheta(theta);
474  fProj->SetPhi(phi);
475  fTheta->Fill(theta, xsec);
476  //slowing of outgoing projectile in target
477  fTarget->SetOutgoing();
478  fTarget->DetectParticle(fProj);
479  //now detect particle in array
480  KVNameValueList* detectors = gMultiDetArray->DetectParticle(fProj);
481  //fill histograms
482  fDepth->Fill(IP.z());
483  FillHistograms(detectors);
484  fProj->GetParameters()->Clear();
485  fProj->SetEnergy(fEnergy);
486  fProj->SetTheta(0);
487  //set random interaction point for scattering
488  if (fIntLayer) {
489  fTarget->SetInteractionLayer(fIntLayer, fBeamDirection);
490  } else {
491  fTarget->GetInteractionPoint(fProj);
492  //if target is multilayer and the interaction layer is not fixed,
493  //the layer & hence the target nucleus may change
494  if (fMultiLayer) {
495  targ_mat = fTarget->GetLayer(fTarget->GetInteractionPoint());
496  fTarg->SetZ((Int_t) targ_mat->GetZ());
497  fTarg->SetA((Int_t) targ_mat->GetMass());
498  fKinematics->SetTarget(fTarg);
499  fKinematics->SetOutgoing(fProj);
500  }
501  }
502  IP = fTarget->GetInteractionPoint();
503  }
504 
505  PrintResults();
506 }

519 void KVElasticCountRates::FillHistograms(KVNameValueList* dets)
520 {
521  // parse the list dets
522  // fill histograms with energy loss for all detectors
523  // clear the detector energy losses
524  // delete the list
525 
526  if (!dets) return;
527 
528  Int_t ndets = dets->GetNpar();
529  for (int i = 0; i < ndets; i++) {
530  TString detname = dets->GetNameAt(i);
531  KVDetector* det = gMultiDetArray->GetDetector(detname);
532  if (!det) continue;
533  TH1F* histo = (TH1F*)fHistos.FindObject(detname);
534  if (!histo) {
535  histo = new TH1F(detname, Form("Eloss in %s", detname.Data()), fBinE, 0, 0);
536  fHistos.Add(histo);
537  }
538  double de = dets->GetDoubleValue(i);
539  histo->Fill(de, xsec * sin(theta * TMath::DegToRad()));
540  histo = (TH1F*)fHistos.FindObject(detname + "_dW");
541  if (!histo) {
542  histo = new TH1F(detname + "_dW", Form("Solid angle of %s", detname.Data()), fBinE, 0, 0);
543  fHistos.Add(histo);
544  }
545  histo->Fill(de, sin(theta * TMath::DegToRad()));
546  TH2F* histo2 = (TH2F*)fHistos.FindObject(detname + "_map");
547  if (!histo2) {
548  histo2 = new TH2F(detname + "_map", Form("Map of %s", detname.Data()), 100, 0, 0, 100, 0, 0);
549  fHistos.Add(histo2);
550  }
551  histo2->Fill(theta, phi, xsec);
552  det->Clear();
553  }
554  delete dets;
555 }

611 void KVElasticCountRates::PrintResults(Double_t beam_intensity)
612 {
613  // Print mean energy deposit & counting rate for given beam intensity in particles per second
614 
615  TIter it(&fHistos);
616  TH1F* h;
617  fRates.clear();
618 
619  std::vector<count_rate> count_rates;
620 
621  while ((h = (TH1F*)it())) {
622  TString name = h->GetName();
623  if (!name.EndsWith("_dW") && !name.EndsWith("_map")) {
624  TH2F* map = (TH2F*)fHistos.FindObject(name + "_map");
625  // integrated cross-section
626  double intXsec = h->Integral() * fVolume / fNtirages;
627  // counting rate
628  double rate = fAtomicDensity * beam_intensity * intXsec;
629  // mean energy
630  double emean = h->GetMean();
631  KVDetector* det = gMultiDetArray->GetDetector(name);
632  double fluence = rate / det->GetEntranceWindowSurfaceArea();
633  double dissipation = emean * rate / det->GetEntranceWindowSurfaceArea();
634  count_rates.push_back(
635  count_rate(name, rate, emean, map->GetMean(), map->GetMean(2), fluence, dissipation, intXsec)
636  );
637  fRates[name.Data()] = KVElasticCountRate(rate, emean, intXsec, fluence, dissipation);
638  }
639  }
640  std::sort(count_rates.begin(), count_rates.end(), compare_count_rates);
641 
642  for (std::vector<count_rate>::iterator it = count_rates.begin(); it != count_rates.end(); ++it) {
643  it->print();
644  }
645 }
