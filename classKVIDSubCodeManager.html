KVIDSubCodeManager Class Reference
The Identification module

Stores concatenations of "subcodes" returned by different identification routines inside a KVIDSubCode (16-bit binary number).

For a given type of identification routine/telescope, the subcode is stored in fNbits bits from bit number fMSB to bit (fMSB-fNbits+1) (fMSB is the bit- number of the most significant bit used to store the subcode - bit 0 is the Least Significant Bit).
