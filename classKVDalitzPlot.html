KVDalitzPlot Class Reference
The Gui module

Fill 3D observables {a1,a2,a3} in a dalitz plot.

, a1+a2+a3 = 1 and fill in a equilateral triangle where the distance from each side of the triangle represent the value of each componment

  • Normalize first the sum to 1 => a1+a2+a3 = 1
  • Deduce the position in an equilateral triangle as following a1n -> distance with the bottom border /\ a2n -> distance with the left border / \ a3n -> distance with the right border /____\
