38 <h2>KVIDGrid</h2>
39 Abstract base class for 2D identification grids in e.g. (dE,E) maps.
40 Such a grid is supposed to be made up of two parts:
41 <ul>
42 <li>a set of lines for the identification i.e. indicating the position of the
43  loci of points corresponding to a given nucleus</li>
44 <li>a set of lines used to indicate the area of the map in which identification
45  is possible (or conversely, the limits beyond which identification is
46  not possible).</li>
47 </ul>
48 <br>
49 Specific implementations of such identification grids must define the following
50 methods :
51 
52  <h4>Bool_t IsIdentifiable(Double_t x, Double_t y) const</h4>
53 
54  - returns either kTRUE or kFALSE for each point (x,y) of the identification
55  map depending on whether such a point can lead to a successful identification
56  or not
57 
58  <h4>void Identify (Double_t x, Double_t y, KVNucleus *nuc) const</h4>
59 
60  - attribute an identification to the KVNucleus for a given point (x,y) in the map.
61  This is most often based on the result of FindNearestIDLine(), which is a general
62  algorithm for finding the ID line which is the closest to a given point.
63 <br>
64 Make sure to call the Initialize() method of the grid once before using it.
65 
66 <h3>Get/SetParameter methods</h3>
67 You can associate any number of parameters with the grid (KVParameter<Double_t> objects)
68 A parameter associates a name and a value. These parameters are used to name the grid.
69 
70 <h3>Grid scaling factors</h3>
71 Get/SetX/YScaleFactor methods can be used to apply global linear scaling factor to grids
72 in one or both dimensions. i.e. if we have a grid pointer
73 
74 <code>
75  KVIDGrid* g;//pointer to a grid
76  g->SetXScaleFactor(1.2);//we apply a 20% scaling factor to the X-coordinates
77  g->Draw(); // X-coordinates of all points in lines are 1.2*original value
78 </code>
79 
80 To remove the scaling factor, call SetX/YScaleFactor() or SetX/YScaleFactor(0).
81 Note that this scaling is taken into account also in IsIdentifiable and Identify methods
82 <!-- */
83 // --> END_HTML

192 KVIDLine* KVIDGrid::NewLine(const Char_t* idline_class)
193 {
194  // Create a new line compatible with this grid.
195  //
196  // If idline_class = "id" or "ID":
197  // create default identification line object for this grid
198  //
199  // If idline_class = "ok" or "OK":
200  // create default 'OK' line object for this grid
201  //
202  // If idline_class = class name:
203  // create line object of given class
204  //
205  // If idline_class = "":
206  // create KVIDLine object
207 
208  TString _cl, _type(idline_class);
209  _type.ToUpper();
210  TClass* clas = 0;
211 
212  if (_type == "ID") clas = DefaultIDLineClass();
213  else if (_type == "OK") clas = DefaultOKLineClass();
214  else _cl = idline_class;
215 
216  if (_cl == "") _cl = "KVIDLine";
217  if (!clas) clas = gROOT->GetClass(_cl.Data());
218 
219  KVIDLine* line = 0;
220 
221  if (!clas) {
222  Error("AddIDLine",
223  "%s is not a valid classname. No known class.", _cl.Data());
224  } else {
225  if (!clas->InheritsFrom("KVIDLine")) {
226  Error("AddIDLine",
227  "%s is not a valid class deriving from KVIDLine.",
228  _cl.Data());
229  } else {
230  line = (KVIDLine*) clas->New();
231  }
232  }
233  return line;
234 }

291 Int_t KVIDGrid::GetIDLinesEmbracingPoint(const Char_t* direction,
292  Double_t x, Double_t y, TList& tmp) const
293 {
294  //Replaces contents of TList 'tmp' with subset of ID lines for which IsBetweenEndPoints(x,y,direction) == kTRUE.
295  //nlines = number of lines in list
296 
297  TIter next(GetIdentifiers());
298  Int_t nlines = 0;
299  KVIDLine* line;
300  tmp.Clear();
301  while ((line = (KVIDLine*) next())) {
302  if (line->IsBetweenEndPoints(x, y, direction)) {
303  tmp.Add(line);
304  nlines++;
305  }
306  }
307  return nlines;
308 }
