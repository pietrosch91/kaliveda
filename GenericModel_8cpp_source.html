18 <h2>BackTrack - a RooFit approach to backtracing</h2>
19 <h4>Backtracing</h4>
20 <p>Backtracing is a statistical procedure for deducing
21 the distributions of model parameters which give the best description of
22 some data which the model is supposed to reproduce by transforming parameter distributions
23 into observable distributions. By backtracing we try to deduce the model parameter distributions
24 from the data observable distributions. See Nuclear Physics A604(1996)183-207
25 and Nuclear Instruments and Methods in Physics Research Section A544(2005)679-683</p>
26 
27 <p>The model is defined by the conditional probability distributions of observables for a given
28 set of parameter values, M(o|p). Then, given a set of parameter distributions P(p) which vary in domain D,
29 the observable distributions are given by
30 <!-- */
31 // --> END_HTML
32 // BEGIN_LATEX
33 // \mathcal{O}(\mathbf{o})=\int_{D_{p}}\mathcal{P}(\mathbf{p})\mathcal{M}(\mathbf{o}|\mathbf{p})\:d\mathbf{p}
34 // END_LATEX
35 // BEGIN_HTML <!--
36 /* -->
37 and the aim of the backtracing procedure is to find the best parameter distributions P(p)
38 which reproduce the data observable distributions O(o).</p>
39 
40 <h4>BackTrack</h4>
41 BackTrack aims to perform backtracing using the <a href="https://root.cern.ch/drupal/content/roofit">RooFit</a> package.
42 RooFit can fit probability density functions to experimental data distributions conserving all correlations
43 between the observables. We discretize the parameter space Dp in order to set up a pseudo-PDF for our model
44 whose adjustable parameters are the weight of each bin in the discretized parameter space. Then by fitting the
45 experimental data with this pseudo-PDF we find the distribution of weights in the parameter space, i.e. the
46 parameter distributions which best describe the data.
47 
48 <h4>Use</h4>
49 <p>In order to perform backtracing/BackTracking, you must write a class which inherits from this class and
50 override the method<br>
51 RooDataSet* <a class="funcname" href="#BackTrack__GenericModel:GetModelDataSet">GetModelDataSet</a>(RooArgList&amp; par)<br>
52 which should create and return a pointer to a RooDataSet containing events generated from the model using the
53 parameter values which are passed in the RooArgList argument. This can be by directly generating the events on demand,
54 or by importing events from e.g. a TTree - whatever you want.</p>
55 
56 <p>Then the basic steps for BackTracking are:
57 <ul>
58 <li> define the parameters for the model with their range and binning using <a class="funcname" href="#BackTrack__GenericModel:AddParameter">AddParameter</a>;
59 <li> define the observables of the data/model with their range using <a class="funcname" href="#BackTrack__GenericModel:AddObservable">AddObservable</a>;
60 <li> generate/import the model observable distributions for each cell in the discretized parameter space using <a class="funcname" href="#BackTrack__GenericModel:ImportModelData">ImportModelData</a>;
61 <li> generate the pseudo-PDF from the imported model distributions using <a class="funcname" href="#BackTrack__GenericModel:ConstructPseudoPDF">ConstructPseudoPDF</a>;
62 <li> fit the pseudo-PDF to the data using <a class="funcname" href="#BackTrack__GenericModel:fitTo">fitTo</a>
63 </ul>
64 See the <a href="BACKTRACK_Index.html">example classes</a> for more details.

251  void GenericModel::ImportModelData(Int_t parameter_num, RooArgList* plist)
252  {
253  // Import all model datasets corresponding to the defined parameters, ranges & binnings
254  RooArgList PLIST;
255  if (plist) PLIST.addClone(*plist);
256  RooRealVar* par = dynamic_cast<RooRealVar*>(fParameters.at(parameter_num));
257  RooAbsBinning& bins = par->getBinning();
258  Int_t N = bins.numBins();
259  RooRealVar* par_in_list = (RooRealVar*)PLIST.find(par->GetName());
260  if (!par_in_list) {
261  PLIST.addClone(RooRealVar(par->GetName(), par->GetTitle(), 0.));
262  par_in_list = (RooRealVar*)PLIST.find(par->GetName());
263  }
264  for (int i = 0; i < N; i++) {
265  par_in_list->setMax(bins.binHigh(i));
266  par_in_list->setMin(bins.binLow(i));
267  par_in_list->setVal(bins.binCenter(i));
268  if ((parameter_num + 1) < GetNumberOfParameters())
269  ImportModelData(parameter_num + 1, &PLIST);
270  else {
271  AddModelData(PLIST, GetModelDataSet(PLIST));
272  }
273  }
274  }

288  void GenericModel::ConstructPseudoPDF(Bool_t)
289  {
290  // Build a parameterised pseudo-PDF from all the imported model datasets.
291  // Each dataset is transformed into a RooNDKeysPdf kernel estimation p.d.f.
292  // This p.d.f. models the distribution of an arbitrary input dataset as a
293  // superposition of Gaussian kernels, one for each data point, each
294  // contributing 1/N to the total integral of the p.d.f.
295  //
296  // If debug=kTRUE, each dataset and kernel PDF will be displayed in a canvas.
297  // This can help to decide if the kernel smoothing parameter needs attention.
298 
299  if (!GetNumberOfDataSets()) {
300  Error("ConstructPseudoPDF", "import model datasets with ImportModelData() first");
301  return;
302  }
303 
304  if (fModelPseudoPDF) {
305  delete fModelPseudoPDF;
306  fModelPseudoPDF = 0;
307  fNDKeys.Delete();
308  fFractions.removeAll();
309  }
310  //generate kernels
311  RooArgList kernels;
312  Double_t initialWeight = 1. / GetNumberOfDataSets();
313 
314  for (int i = 0; i < GetNumberOfDataSets(); i++) {
315 
316  Info("ConstructPseudoPDF", "Kernel estimation of dataset#%d...", i);
317  RooNDKeysPdf* p = new RooNDKeysPdf(Form("NDK%d", i),
318  Form("Kernel estimation of dataset#%d", i),
319  GetObservables(),
320  *((RooDataSet*)fDataSets[i]),
321  "am", fSmoothing);
322  fNDKeys.Add(p);
323  kernels.add(*p);
324  if (i < (GetNumberOfDataSets() - 1))
325  fFractions.addClone(RooRealVar(Form("W%d", i), Form("fractional weight of kernel-PDF #%d", i), initialWeight, 0., 1.));
326 
327  }
328  fModelPseudoPDF = new RooAddPdf("Model", "Pseudo-PDF constructed from kernels for model datasets", kernels, fFractions, kTRUE);
329  }

337  RooFitResult* GenericModel::fitTo(RooDataSet* data)
338  {
339  // Perform fit of the pseudo-PDF to the data
340  // On multi-core machines, this automatically uses all available processor cores
341 
342  SafeDelete(fLastFit);
343 #ifdef WITH_MULTICORE_CPU
344  fLastFit = fModelPseudoPDF->fitTo(*data, Save(), NumCPU(WITH_MULTICORE_CPU));
345 #else
346  fLastFit = fModelPseudoPDF->fitTo(*data, Save());
347 #endif
348 
349  SafeDelete(fParamDataHist);
350  fParamDataHist = new RooDataHist("params", "params", GetParameters());
351 
352  // store weights of component pdfs => distribution of parameters
353  fWeights.removeAll();
354  const RooArgList& coefs = fModelPseudoPDF->coefList();
355  for (int i = 0; i < GetNumberOfDataSets(); i++) {
356  RooAbsReal* coef = (RooAbsReal*)coefs.at(i);
357  RooRealVar w(Form("w%d", i), Form("Fitted weight of kernel#%d", i), coef->getVal());
358  if (coef->InheritsFrom(RooRealVar::Class())) {
359  w.setError(((RooRealVar*)coef)->getError());
360  } else {
361  w.setError(coef->getPropagatedError(*fLastFit));
362  }
363  fWeights.addClone(w);
364  fParamDataHist->set(*GetParametersForDataset(i), w.getVal(), w.getError());
365  }
366 
367  SafeDelete(fParameterPDF);
368  fParameterPDF = new RooHistPdf("paramPDF", "paramPDF", GetParameters(), *fParamDataHist);
369 
370  return fLastFit;
371  }
