219  {
220  // The insert mode determines how to deal with errors caused by constraints
221  // The possible values of i are (with corresponding SQLite meaning):
222  // KVSQLite::insert_mode::DEFAULT ("INSERT INTO [table] ...")
223  // KVSQLite::insert_mode::FAIL ("INSERT OR FAIL INTO [table] ...")
224  // - if any data in row being inserted fails a constraint on one or more
225  // columns, insertion fails with error
226  // KVSQLite::insert_mode::IGNORE ("INSERT OR IGNORE INTO [table] ...")
227  // - if any data in row being inserted fails a constraint on one or more
228  // columns, the row is silently ignored (existing row unchanged)
