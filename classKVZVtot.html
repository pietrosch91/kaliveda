KVZVtot Class Reference
The Globvars module

Global variable returning the ZVtot variable. It is defined as follow:

ZVtot= Sum(Z(i)*v(i))

with v(i) : velocity of particle i; (in cm/ns) Z(i) : charge of particle i;

The units of ZVtot are therefore [cm/ns].

By default ZVtot is calculated from particle velocities in the default (laboratory) frame. However, this can be changed by giving the name of the desired frame to either the constructor KVZVtot* zvtot = new KVZVtot("my_ZVtot", "CM"); // calculate in "CM" frame or by using KVVarGlob method SetFrame(): zvtot->SetFrame( "QP" ); // calculate in "QP" frame

The names of the variables and the corresponding indexes are as follows: 

 Name            Index                   Meaning

Z 0 (default) Z component of ZVtot X 1 X component of ZVtot Y 2 Y component of ZVtot

All these values can be obtained by calling the GetValuePtr() method which returns an array of Double_t containing the values.

The vector (TVector3) can be used by calling the GetTVector3() method.

If you want to normalize the values e.g. to the product Z*v of the projectile in the laboratory frame, set the parameter "Normalization" Int_t KVZVtot::nb = 0; Int_t KVZVtot::nb_crea = 0; Int_t KVZVtot::nb_dest = 0;
