KVNuclDataTable Class Referenceabstract
The Particles module

ABC for nuclear table.

General methods:

  • Bool_t IsInTable(Int_t zz, Int_t aa) - returns kTRUE for nuclei which appear in the read table, kFALSE for others

    Specific implementations must define the methods:

    • Initialize() - called once to initialize mass table, for example by reading in values from a file
    • Double_t GetValue(Int_t zz, Int_t aa) - returns value of the tabulated quantity
    • CreateTable(Int_t ntot) - Create table where for each nucleus, a pointeur of the class will be stored
    • CreateElement(Int_t idx)- Create element which correspond to the class pointeur associated to the nucleus For example : If we consider a class KVLifeTime where is stored the value of life time of a nucleus
      void KVNuclDataTable::CreateTable(Int_t ntot) { theTable = new KVLifeTime*[ntot]; }
      void KVNuclDataTable::CreateElement(Int_t idx) { current_idx = idx; theTable[idx] = new KVLifeTime(); } For further detail see the KVLifeTimeTable and KVLifeTime class
