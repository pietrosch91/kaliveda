KVMFMDataFileReader Class Reference
The Daq_cec module

Read MFM format acquisition data.

This class uses the mfmlib package available from: https://gitlab.in2p3.fr/jdfcode/mfmlib.git This class is enabled if you build kaliveda with cmake option -DUSE_MFM=yes
