The Geometry module

Base class for calibrating detectors. ********

Each KVCalibrator has a set of parameters (coefficients), possibly read from a file at initialisation time. The actual functional form of the calibration formula using these coefficients is defined in the child classes of KVCalibrator via the following methods : Double_t Compute(Double_t x,....) ou Double_t operator()(Double_t x,....)
