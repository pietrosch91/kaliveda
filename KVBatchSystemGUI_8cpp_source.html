21 //Graphical interface for monitoring, altering, and deleting batch jobs.
22 //To run:
23 //
24 // root[0] new KVBatchSystemGUI
25 //
26 //It will automatically connect to the default batch system and display
27 //all of the user's queued or running jobs along with information
28 //on resource requests and consumption.
29 //The buttons at the top of the GUI can be used to (manually) refresh
30 //the informations (see Automatic refresh, below), alter the properties
31 //of selected (queued) job(s), or delete selected (queued or running) job(s).
32 //
33 //Automatic refresh
34 //The GUI automatically refreshes every N seconds, as defined by the
35 //following environment variables (.kvrootrc):
36 //
37 // KVBatchSystemGUI.AutoUpdate: yes
38 // KVBatchSystemGUI.RefreshInterval: 30
39 //
40 //Change these variables in your .kvrootrc if you want to modify or
41 //disable automatic refreshing.
