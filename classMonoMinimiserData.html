The Minimiser module

MonoMinimiserData.cpp

Description

Class for handling data produced by the SiliconEnergyMinimiser. The minimiser function usually just returns the A value it has determined to be the most likely for a given Z (i.e. It returns an integer). Therefore to obtain more interesting information about how well the algorithm is performing, we need to pass in an (optional) pointer to a data class such as this one for the minimiser implementation to populate.

This class handles the extra data produced by the MonoMinimiserImpl implementation, in addition to the basic minimiser data (See MinimiserData).
