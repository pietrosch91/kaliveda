21 // Energy limits
22 // Normally all range, dE, Eres functions are limited to range 0<=E<=Emax,
23 // where Emax is nominal max energy for which range tables are valid
24 // (usually 400MeV/u for Z<3, 250MeV/u for Z>3)
25 // If higher energies are required, call static method KVedaLoss::SetIgnoreEnergyLimits()
26 // BEFORE ANY MATERIALS ARE CREATED
27 // in order to recalculate the Emax limits in such a way that:
28 // range function is always monotonically increasing function of Einc
29 // stopping power is concave (i.e. no minimum of stopping power followed by an increase)
30 // at the most, the new limit will be 1 GeV/nucleon.

246 TObjArray* KVedaLoss::GetListOfMaterials()
247 {
248  // Create and fill a list of all materials for which range tables exist.
249  // Each entry is a TNamed with the name and type (title) of the material.

251

261  }
262  return 0;
