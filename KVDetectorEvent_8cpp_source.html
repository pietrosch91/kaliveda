27 //KVDetectorEvent
28 //
29 //Multiparticle event detected/"seen" by a multidetector array.
30 //The result is basically a list of all the hit groups of the array (see KVGroup and KVMultiDetArray), from which we
31 //can access the energy losses etc. in the different detectors.
32 //
33 //A group is said to be "hit" if at least one detector has fired in the group (see KVDetector::Fired()).
34 //The list of "hit" groups is filled by KVMultiDetArray::GetDetectorEvent().
35 //
36 //GetGroups() returns a pointer to the list of hit groups.
37 // To loop over all hit groups in a KVDetectorEvent* kde, do something like this:
38 // KVGroup *g;
39 // TIter next( kde->GetGroups() );
40 // while( (g = (KVGroup*)next()) ){
41 // ...
42 // }
43 //
44 //ContainsGroup(KVGroup* grp) returns kTRUE if 'grp' points to a KVGroup belonging to the list of hit groups
45 //i.e. if ContainsGroup() is equal to kTRUE, then the group was hit in the event.
