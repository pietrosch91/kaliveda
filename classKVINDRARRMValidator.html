KVINDRARRMValidator Class Reference
The Identification module

Utility base class used to define validity of various INDRA-related objects (identification grids, calibration parameters, etc. etc.) in terms of:

  • a list of runs
  • a list of rings
  • a list of modules Each list is kept in an object of the KVNumberList, which handles very general lists such as "1-3, 7, 10-19, 21". KVINDRARRMValidator::KVINDRARRMValidator() {
