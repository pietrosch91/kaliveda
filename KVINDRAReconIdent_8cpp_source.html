59  // ReconIdent.DataAnalysisTask.OutputRepository: [name of repository]
60  // [name of dataset].ReconIdent.DataAnalysisTask.OutputRepository: [name of repository]
61  //
62  // If no value is set for the current dataset (second variable), the value of the
63  // first variable will be used. If neither is defined, the new file will be written in the same repository as
64  // the recon file (if possible, i.e. if repository is not remote).
65 
66  // get dataset to which we must associate new run
